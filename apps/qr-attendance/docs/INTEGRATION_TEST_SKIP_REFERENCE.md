## 条件付きで SKIP になる項目（createdEventId がないとき）

4.1.1（イベント作成）が失敗し、**createdEventId が取得できない**場合、以下の項目は実行されず **SKIP** 扱いになります。

| No | カテゴリ | 項目名 | APIエンドポイント |
|----|----------|--------|-------------------|
| 4.2.1 | イベント | 更新（正常） | PUT /v1/admin/events/{eventId} |
| 4.2.2 | イベント | 削除（正常） | DELETE /v1/admin/events/{eventId} |
| 4.3.1 | イベントQR | QRデータ取得 | GET /v1/admin/events/{eventId}/qr |
| 4.3.2 | イベントQR | 権限外 | GET /v1/admin/events/{eventId}/qr |
| 5.1.1 | 参加申込 | 申込（正常） | POST /v1/users/events/{eventId}/register |
| 5.1.2 | 参加申込 | 二重申込 | POST /v1/users/events/{eventId}/register |
| 5.1.4 | 参加申込 | 取消 | DELETE /v1/users/events/{eventId}/register |
| 6.1.1 | 打刻 | 打刻（正常） | POST /v1/users/attendance |
| 6.1.2 | 打刻 | 未申込イベント | POST /v1/users/attendance |
| 8.1.1 | レポート | 出席CSV出力 | GET /v1/admin/reports/events/{eventId}/csv |
| 8.1.2 | レポート | 権限外 | GET /v1/admin/reports/events/{eventId}/csv |

※ スプレッドシートの 8.1.1 / 8.1.2 のエンドポイントは「/v1/admin/reports」と記載されていますが、実装は **GET /v1/admin/reports/events/{eventId}/csv** です。
